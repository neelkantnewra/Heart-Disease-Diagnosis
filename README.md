## To run the file


`git clone https://github.com/neelkantnewra/Heart-Disease-Diagnosis.git`


For linux:

`python3 -m venv hsd-venv`

`source hsd-venv/bin/activate`

`pip install -r requirements.txt`

`flask run`

For Windows:

`python -m venv hsd-venv`

`hsd-venv\Scripts\activate`

`pip install -r requirements.txt`

`flask run`
